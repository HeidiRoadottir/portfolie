Brøndby IF – Data-Driven Football Analytics

Bachelor project · Sports analytics · Statistics · R

This project explores how data can support strategic and tactical decision-making in Brøndby IF. Using detailed event data from matches, we built an Expected Goals (xG) model and additional analyses to understand shot quality, patterns of play and player behaviour.

R Predictive modelling Clustering Sports analytics Visualisation

Context & challenge

Football decisions are often driven by emotion, intuition and tradition. At the same time, clubs increasingly have access to rich data about every pass, shot and position on the pitch. The challenge was to:

  • Build a meaningful xG model for Brøndby IF using event data.
  • Explore how different variables influence shot success.
  • Translate analytical results into insights that coaches could use.

My role

  • Loaded and merged multiple datasets of Superliga match events in R.
  • Prepared features describing shot context (distance, angle, body part, etc.).
  • Helped train and evaluate the xG model on training/test splits.
  • Created visualisations and descriptive statistics to communicate findings.

Modelling & analysis

Feature engineering

We engineered variables including:

  • Shot distance and angle to goal.
  • Body part used (foot, head, other).
  • Match context, such as half and game state.
  • Team and opposition formations.

xG model

Using these features, we trained a model to estimate the probability that a shot becomes a goal. We then validated it on a held-out test set to evaluate performance and calibration.

Additional insights

Beyond the core xG model, we explored passing patterns, shot locations and player clusters to highlight where Brøndby created high-quality chances and where they struggled.

Outcome & reflections

The project demonstrated that:

  • xG provides a more stable performance measure than raw goals.
  • Spatial features (distance, angle) are strong predictors of success.
  • Visualisations help make complex models understandable for non-technical stakeholders.

For me, this project solidified my interest in data analysis and modelling, and gave me experience applying these skills in a domain that is both complex and very concrete.
